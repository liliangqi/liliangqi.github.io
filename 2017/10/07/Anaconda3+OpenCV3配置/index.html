Anaconda3+OpenCV4配置

2017-10-07 machine learning

　　随着人工智能和机器学习的浪潮扑面而来，Python以其语法简单、使用灵活、解释性强的优点也逐渐变得炙手可热。然而使用Python实现的许多科学计算、绘画制图、数据处理往往依赖于Python社区强大的第三方库，比如NumPy，Matplotlib等等。我们在使用这些库时不免要提前进行安装，而且往往安装不全，发生缺少某库的错误，令人头痛不已。

　　Anaconda作为一个Python的集成环境版本，包含了大部分我们平时工作学习所需要用到的Python第三方库，而且易于管理和扩展，故笔者将在此博客中介绍各大平台上Anaconda的安装，并针对计算机视觉工作者介绍OpenCV的安装。

　　此外，虽然目前很多从业者仍在大量使用Python2，而且官方也表示会继续支持Python2的维护，但Python的未来一定是更新的Python3，所以我们也只介绍安装对应的Anaconda3。

MacOS环境

　　macOS自带Python，但其版本为2.7.6，而且不含我们所需的大多数第三方库，故接下来介绍macOS上Anaconda3的安装。

##安装Anaconda3

  1. 进入Anaconda下载官网，选择macOS，然后选择Python 3.6版本

  2. 这里有两种安装方式，一种是图形界面安装，另一种是命令行安装，这里我们选择图形界面安装，点击绿色的图标Download

    ​

  3. 下载完成后打开Anaconda3-5.0.0-MacOSX-x86_64.pkg

  4. 按照安装提示完成安装，无需更改目的卷宗，它会自动安装到你的主文件夹下

  5. 安装完成后我们能够发现在~/.bash_profile文件中增加了两行

    # added by Anaconda3 4.4.0 installer
export PATH="Users/yourname/anaconda/bin:$PATH"

    这里是完成了环境变量的配置，要使其生效，则在终端输入source ~/.bash_profile即可。

  6. 在终端中输入python，弹出如下信息则说明安装成功。

安装OpenCV3

　　macOS上安装Anaconda2对应的OpenCV比较简单，直接使用conda install opencv3即可，但在最新的Anaconda3对应的Python3.6的官方包里还没有OpenCV的支持，因此目前使用conda forge的包。

  1. 添加conda forge的频道

    conda config --add channels conda-forge

  2. 安装OpenCV及其相应的依赖（依赖会自动安装）

    conda install opencv

Windows环境

　　Windows原生系统不带Python，我们直接进行Anaconda3的安装。

##安装Anaconda3

  1. 进入Anaconda下载官网，选择Windows，然后选择Python 3.6版本

  2. Windows对应的安装方式只有图形界面安装，根据自己计算机和系统的位数选择32位或64位安装包下载即可

  3. 打开刚才下载的Anaconda3-5.0.0-Windows-x86_64.exe

  4. Destination Folder无需设置，默认为C盘，如有需要可进行更改，但要记住安装目录

  5. 勾选以下两项，将Anaconda3加入环境变量并将Anaconda3作为默认的Python3.6

  6. 使用cmd命令呼出终端界面，输入python，弹出如下信息则说明安装成功。

##安装OpenCV3

　　Windows上安装OpenCV3的方法与macOS上类似，仍然是用如下的两行命令进行安装。

conda config --add channels conda-forge
conda install opencv
