Python基础之：函数

2017-07-05 Python基础

　　函数是带名字的代码块，用于完成具体的工作。

　　要执行函数定义的特定任务，可调用该函数。需要在程序中多次 执行同一项任务时，无需反复编写完成该任务的代码，而只需调用执行该任务的函数，让Python运行其中的代码。通过使用函数，程序的编写、阅读、测试和修复都将更容易。

传递列表

在函数中修改列表

  • 在函数中对参数传递的列表所做的任何修改都是永久性的

禁止函数修改列表

  • 调用函数时将列表的副本（切片）而非本身传递给函数

    function_name(list_name[:])

  • 非必要的情况下还是尽可能向函数传递原始列表，这样可以避免花时间和内存创建副本，从而提高效率

传递任意数量的实参

  • 在形参前加*即可
  • 调用函数时传递的实参则会被封装到一个元组中

结合使用位置实参和任意数量实参

  • 若要让函数接受不同类型的实参，必须在函数定义中将接纳任意数量实参的形参放在最后

    def make_pizza(size, *toppings):

使用任意数量的关键字实参

  • 关键字形参前加**即可

  • 关键字形参可以理解为字典的键，关键字实参可以理解为键对应的值

    eg: 形参\**user_info让Python创建一个名为user_info的空字典，并将收到的所有名称-值对都封装到这个字典中

将函数存储在模块中

##导入整个模块

  • import module_name
  • 调用函数方法：module_name.function_name()

导入特定的函数

  • from module_name import function_name
  • from module_name import function_0, function_1, function_2
  • 调用函数时无需使用句点：function_name()

使用as给函数指定别名

  • from module_name import function_name as fn
  • 调用时直接使用fn()

使用as给模块指定别名

  • import module_name as mn
  • 调用时使用mn.function_name()

导入模块中的所有函数

  • from module_name import *
  • 调用函数时无需使用句点：function_name()
