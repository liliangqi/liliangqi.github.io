Joint Detection and Identification Feature Learning for Person Search数据集

2017-07-06 machine learning

　　这篇文章主要总结了《Joint Detection and Identification Feature Learning for Person Search》一文中所用到的数据集的结构，帮助理解训练以及测试方式。

./Image/SSM

包含所有的18184张图片，包括街道拍摄和电影截屏

./annotation/Images.mat

这是一个1*18184的结构体，其中包含三个变量

  • imname: 图片名
  • nAppear: 包含的行人个数
  • box: 1*nAppear的结构体变量，表示nAppear个行人的位置
    • idlocate: 每个行人位置对应的四个坐标
    • ishard: 取值为0或1，用途未知

./annotation/Person.mat

这是一个1*11934的结构体，其中包含三个变量

  • idname: identity的编号（字符串变量）
  • nAppear: 该identity出现的次数
  • scene: 1*nAppear的结构体变量，表示该identity出现的图片及位置
    • imname: 图片名
    • idlocate: identity对应的instance的坐标
    • ishard: 取值为0或1，用途未知

./annotation/pool.mat

这是一个1*6978的元组，包含了6978个测试图片的文件名（字符串变量）

./annotation/test/train_test/Train.mat

这是一个1*5532的元组，每个元素是一个1*1的结构体，以下为结构体的结构

  • idname: identity的编号（字符串变量）
  • nAppear: 该identity出现的次数
  • scene: 1*nAppear的结构体变量，表示该identity出现的图片及位置
    • imname: 图片名
    • idlocate: identity对应的instance的坐标
    • ishard: 取值为0或1，用途未知

./annotation/test/train_test/TestG50.mat

这是一个1*2900的结构体，其中包含两个变量

  • Query: 1*1的结构体变量，表示用作query的instance
    • imname: 该instance出现的图片的文件名（字符串变量）
    • idlocate: 该instance的坐标信息
    • ishard: 取值为0或1，用途未知
    • idname: 该instance所属的identity的编号（字符串变量）
  • Gallery: 1*50的结构体变量，表示用作gallery的图片信息
    • imname: 图片名
    • idlocate:
      • 若该图片中包含对应identity的instance，则为其对应的坐标信息
      • 若该图片中不包含对应identity的instance，则该变量为空矩阵
    • ishard:
      • 若该图片中包含对应identity的instance，则为其取值为0或1，用途未知
      • 若该图片中不包含对应identity的instance，则该变量为空矩阵

TestG100~TestG400.mat等文件类似。
